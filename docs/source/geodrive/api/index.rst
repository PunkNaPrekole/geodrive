Описание модулей
----------------

**clients**
    Основные клиенты для работы с ровером. Используйте эти классы для управления роботом.

**schemas**
    Модели данных для телеметрии, команд и статусов.

**commands**
    Доступные команды для управления ровером.

.. note::
   Для работы с SDK используйте только клиенты из модуля ``clients``.
   Остальные модули предназначены для внутреннего использования.
